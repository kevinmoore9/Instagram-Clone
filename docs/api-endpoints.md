# API Endpoints

## HTML API

**Root**
* GET / - loads React web app

## JSON API

**Users**
* POST /api/users
* PATCH /api/users
* GET /api/users/:username

**Sessions**
* POST /api/session
* DELETE /api/session

**Photos**
* GET /api/photos
* GET /api/users/:photoId
* POST /api/photos
* PATCH /api/photos/:photoId
* DELETE /api/photos/:photoId
