# API Endpoints

## HTML API

**Root**
* GET / - loads React web app

## JSON API

**Users**
* POST /api/users
* PATCH /api/users
* GET /api/users/:username

**Sessions**
* POST /api/session
* DELETE /api/session

**Images**
* GET /api/images
* GET /api/users/:imageId
* POST /api/images
* PATCH /api/images/:imageId
* DELETE /api/images/:imageId
